Remove Locations

Use this to remove locations from an organization, e.g. if a branch closes. It requires an organization ID to remove from (a sanity check, the backend knows from the location ID what organization it belongs to). It also requires the location(s) to remove. This requires at least one location to move and has no limit on how many. It is the HTML front end I wrote that imposes the limit of 5.
